基于 vector<string> 的命令行数据管理工具，支持增删改查及文件持久化。

编译运行
g++ -o repo main.cpp -std=c++11
./repo
操作流程
启动后选择：1-读取存储库，2-新建存储库

进入主菜单进行数据操作

退出时自动保存

主菜单选项
增加项（连续添加，输入 exit 退出）
删除项（按索引删除）
修改项（按索引修改）
查找项（返回第一个匹配项）
模糊查找（显示所有匹配项）
读取存储库（切换文件）
新建存储库（清空并创建新文件）
退出（自动保存）
存储格式
文件为 .txt 格式，每行：索引 数据内容

注意事项
存储库名称不含空格，自动添加 .txt 后缀

索引从 0 开始

创建已存在的存储库会询问是否覆盖

退出时自动保存，无需手动保存
